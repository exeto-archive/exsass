# exsass

> A small mixin library for Sass.

**THIS PACKAGE IS NO LONGER MAINTAINED**

## Install

```bash
$ npm install --save exsass
```

## Usage

```scss
@import 'node_modules/exsass/exsass';

// or if you use sass-loader

@import '~exsass';

.something {
  @extend %ex-clearfix;
}
```
